Pip install and PyPi

- PyPi is a repository for open-source third-party Python packages
- pip can be used to install packages from the command line
    - pip install package_name

===================================================================================

Modules and packages

- Modules are just .py script that you call in another .py script
- Package are collection of Modules
- __init__.py tells the python that the folder has some module in it
